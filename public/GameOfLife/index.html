Click on each square to create or destroy a cell. Click on Start to run the simulation. Alternatively, you can use a script into the textbox to load a simulation. Click on Generate to get the script for the current frame of the simulation.

Available instructions:

  • grid [size] > Initialize the grid. Each square will be [size] pixels long.
  • add [x][y] > Adds a cell at the coordinates [x] and [y].
  • rem [x][y] > Removes a cell at the coordinates [x] and [y].
  • block [x1][y1][x2][y2] > Creates a block of cells from [x1][y1] to [x2][y2]
  • hollow [x1][y1][x2][y2] > Creates a hollow block of cells from [x1][y1] to [x2][y2]
  • line [x1][y1][x2][y2] > Creates a line of cells from [x1][y1] to [x2][y2]
  • You can comment a line using a # at the beginning of the line.
