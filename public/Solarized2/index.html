What is Solarized

Solarized is a color palette designed for use with terminal and GUI applications, originaly created by Ethan Schoonover. However, this solarized CSS has been adapted to look good with websites. This CSS includes classes for common HTML tags (up to version 5), including tables, inputs, etc, as well as some extra classes to quickly style your divs.

Some extra CSS documents are included for special controls, and CSS menus (up to 3 levels). This means that you can also have awesome controls for your techy web applications! Don't you believe me? Here, have a sample!

Techy form

It's very easy to use these elements. All you need is to include solarized2.css and solarized-radios.css, and you are ready to go! Make sure you have a folder named 'img' with a file called 'checks.png' inside with the graphics for the buttons.

Easy CSS menus!

The CSS template includes files for horizontal and vertical menus. These menus are made out of pure CSS, no javascript required! All you need to do to add one of these awesome menus to your solarized website, is to include solarized-hmenu.css (horizontal) or solarized-vmenu.css (vertical), and then create a nav with a unorganized (ul) list with links (a) inside.

Each li element can contain another ul (after the a link) for a dropdown submenu. It's that easy! See an example:

Level 3 dropdowns (ul) have left and right classes to manually align them to the left or the right depending on their position of the screen.

Quick format classes

Solarize your divs and texts really quick with the premade solarize classes! Classes for texts start with solarize-tone, while classes for divs start with solarize. Take a look at the available classes:
